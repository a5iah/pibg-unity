# pibg-unity

PIBG Management System - From PIBG, For PIBG, By PIBG

## Modules
- Security
- User and Role Management
- Member Management
- Communication Management
- Event Management
- Budget Management
- Procurement Management
- Expenses Management
- Accounting
- Reporting

### Security
- Allow OAuth2/OIDC with 3rd party (Google, private OAuth2/OIDC)

### User and Role Management
- Member
- Parent
- Teacher
- School Management
- AJK
- Super Admin
- School/PIBG Manager
- School/PIBG Admin
